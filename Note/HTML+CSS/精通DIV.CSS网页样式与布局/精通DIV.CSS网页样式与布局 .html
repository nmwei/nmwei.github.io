第一章 开发网站，我们需要做些什么

1. 网页是一个文件，存储在某一台与互联网相连的计算机或服务器中，经由统一资源定位器(URL)来识别和存取。 

2. web （互联网总称）web的本意是蜘蛛网和网的意思，在网页设计中我们称为网页的意思。现广泛译作网络、互联网等技术领域。表现为三种形式，即超文本（hypertext）、超媒体（hypermedia）、超文本传输协议（HTTP）等。

3. 在用HTML语言开发网站之前，需要请美工用Photoshop设计出每个网页的效果，一般出样的文件格式是PSD。

4. CSS主要用于定义网页中的各部分及元素的样式，比如图片的大小、位置、边框的样式等等。而JavaScript主要定义网页的动态效果，与鼠标键盘或定时器有关的动作。

5. 网页主要由三部分组成：结构(Strucure)、表现(Presentation)和行为(Behavior)。

第二章 HTML和XHTML

1. XHTML是HTML的扩展，表示可扩展的超文本标记语言(Extensible HyperText Markup Langugae)。

2. HTML语法指定图片和它周围文本对齐方式:

<img src=”” align=”top/center/bottom/left/right” />

<p> <h> <img>三个标签在html语法中都有align属性。而<a>标签没有align属性。

注意：<img>元素在html语法中有align属性，设置图片与周围文本的对其方式，并且当align取值为center时，表示文字在图片的垂直居中位置，而不是图片在页面的水平居中位置。<img>元素在CSS语法中没有text-align属性，如要设置图片在页面水平居中，需要将图片嵌套在<div>或者<td>中，区别在于如果嵌套在div中，无需设置div宽度，div宽度属性默认100%，而若嵌套在td中，一定要设置table的宽度为100%。

table是块级元素，默认占一整行，但table在未设宽高时，宽度和高度都会自动收缩到能包容下内容的最小高度和宽度。

设置图片在页面水平居中的最简便方法，用<center>标签嵌套图片。

如果设置<a>标签在页面水平居中，可将a嵌套在无序列表li中，在CSS语法中设置ul的text-align属性，或在HTML语法中设置ul的align属性。

3. <table border=””></table>设置边框宽度。

4. div标签不能嵌套在p标签中。在div标签中可以包含文字、图像、表格等标签，用其属性align来设置其中内容的对其方式，取值为left、center、right。

 注意: <a>标签没有align属性，而<a>标签若嵌套在<div>标签中，则可用align属性设置对齐方式。

第三章 CSS基础知识介绍

1. 通过在CSS里编写选择器的嵌套代码，能大大减少对class和id的声明。具体的做法是：通过只给外层标记定义class或者id来引入CSS样式；内层标签则是通过嵌套表示来引入样式，而不需要再定义新的class和id。（外层标记内层嵌套）

2. 属性选择器：元素[属性^=”属性值”]匹配具有此属性，并以此属性值作为开头的元素。元素[属性$=”属性值”]匹配具有此属性，并以此属性值作为结尾的元素。元素[属性*=”属性值”]匹配具有此属性，并含有此属性值的元素。

例：p[align^=”center”]{color:#000}

第四章 通过CSS控制字体样式

1. 设置文字阴影的颜色值：{text-shadow:12px 12px 12px #FF0000;} ，IE9以上版本浏览器支持。

2. 设置字体类型：{font-family：”隶书”}，注意字体文字要加引号。

3. 设置字体粗细：{font-weight：100-900/bold/bolder/lighter/normal;},浏览器默认的粗细是400。

4. 设置文字倾斜：{font-style:inherit/italic/normal/oblique} 继承/斜体/正常/偏斜体。

5. 定义字体大小：{font-size:设置px;}    

注意：px不能省略，否则不生效。而定义图片宽高时width:100px可省略。

注意：设置字体颜色是{color:颜色值}而不是{font-color:颜色值}。CSS语法中没有font-color属性。

6. 定义行高：{line-height:数值px;}

7. 下划线、顶划线与删除线：

{text-decoration:inherit/none/underling/overline/line-through/blink}注：blink使文本闪烁。

8. 设置字间距：{letter-spacing:ex/em/+ex/-ex/+em/-em}

9. 控制文本对其方式：{text-align:left/right/center/justify}

注：在CSS语法中文本对齐方式是text-align而不是align。CSS语法中没有align属性。

10. 定义文本溢出表现形式：{text-overflow:clip/ellipsis/ellipsis-word}

clip裁切；ellipsis显示省略标记；cllipsis-word显示省略标记，插入到最后一个词。

注意：text-overflow属性如果想生效，需要与white-space:nowrap禁止换行和overflow:hidden禁止文本溢出显示共同作用。

11. 定义内容到达容器边界时的表达方式：

{word-wrap:normal/break-word} normal:顶破   break-word：换行

第五章 通过CSS定义链接样式

1. 多个超链接组成导航栏，导航栏用无序列表嵌套超链接标签构成。

2. CSS伪类：未访问a:link 悬停a:hover 激活a:active 已访问a:visited

1.给超链接文本添加提示文字：a{title:提示文字}；给图片添加提示文字：<img href=”” alt=””  title=”提示文字”/>。

3. 定义鼠标特效：{cursor:crosshair十字鼠标;pointer点状鼠标;text文字鼠标;wait等待鼠标;help求助鼠标;……}。

4. 图片不重复。{background-repeat:no-repeat}。

第六章 通过CSS定义图片样式

1. 定义图片边框

在html语法中为<img src=”” border=”数值”;>，设置边框宽度。 

CSS语法中为{border-style:参数;border-color:参数;border-width:参数;}，分别设置边框样式（“点划线”“虚线”等）、边框颜色、边框宽度。

2. 定义图片大小

img{width:数值; height:数值;}    <img src=”” width=”数值” height=”数值”;>

注意：设置图片尺寸用html语法和用CSS语法基本相同。

定义图片最大宽度：img{max-width:最大宽度值;}

3. 定义图片对其方式

横向对齐：图片的横向对齐和文字的横向对齐方法相似，有“左中右”三种。图片本身没有text-align属性，需要通过“外层属性”的对齐方式，定义内层属性的图片对其方式。例：把图片放在表格中，定义表格<td>的text-align属性。

定义文字水平居中，同样是给外层元素（<p>、<div>）定义的text-align,通过“CSS属性继承”的方式来实现文字横向对齐方式。

注意：HTML语法指定图片和它周围文本对齐方式:

<img src=”” align=”top/center/bottom/left/right” />

纵向对齐：{vertical-align:sub/super/top/middle/bottom/text-top/text-bottom;}

4. 设置文字环绕方式

{float:left/right/none;}   注意：没有center。

{padding:数值1 数值2 数值3 数值4;} 

定义图片上、右、下、左四个方向内边距，设置图片和文字之间的间距。

5. 定义元素的固定定位  {position:fixed;}

6. 设置背景颜色：{background-color:颜色参数;} 处理背景图像样式： 

{background-image:url(图片图径);background-repeat:repeat-x/repeat-y/no-repeat;}

注意：设置背景颜色和设置背景图像，可作用在body和div上。(选择器不是img)

定义背景图片位置：{background-position:top/center/bottom  left/center/right;}

固定背景图片：{background-attachment:fixed/scroll;}

注意：在CSS语法中，position是设置元素固定定位。background-position是设置背景图片的位置。background-attachment是设置背景图片是否固定。

第七章 通过CSS设置表格的样式

1. 虽然存在使用DIV替换表格的趋势，但表格在网页开发过程中依然发挥着不可替代的作用。

2. 表格同样可以设置背景颜色：table:{background-color:# }

3. 表格的border、cellspacing、cellpadding都是作用在table的HTML属性，而不是CSS属性。例如：<table border=”2” cellspacing=”3” cellpadding=”3”></table>注意表格的border属性与cellspacing属性属性的区别：border属性是设置整个表格的边框，而cellspacing是设置表格中相邻单元格之间的间距，cellpadding是设置表格中单个单元格与其中文本内容的间距。需要注意的是，单元格中的文本内容默认在单元格td的垂直中央位置（水平左对齐位置），在单元格th的垂直中央位置（水平居中位置）。

4. border、cellspacing、cellpadding都是作用在table的HTML属性，而不是CSS属性。colspan、rowspan是作用在td的HTML属性，而不是CSS属性。

5.  表格中<thead></thead>之间的<tr></tr>表示页眉，<tbody></tbody>之间的<tr></tr>表示主体<tfoot></tfoot>之间的<tr></tr>表示页脚。
